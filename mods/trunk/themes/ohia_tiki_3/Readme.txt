* __Description__: Ohia is a dark red and gray theme using something like the colors of the blossom of the ohia lehua tree and the lava where ohia is among the first new life seen after the lava cools. Ohia uses the *litecss source-ordered layout method ([http://sourceforge.net/projects/litecss]). This package is for use with Tiki 3.0. Previous Ohia template (.tpl) files should be removed and replaced with these versions.
