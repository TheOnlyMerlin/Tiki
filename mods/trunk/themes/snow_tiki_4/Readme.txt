* __Description__: This is an original theme for Tiki CMS (http://tikiwiki.org). It features snowflake-pattern graphic backgrounds for module boxes and other page components, and has an optional Flash title graphic (not tested with the version of the theme). Snow uses the *litecss source-ordered layout method ([http://sourceforge.net/projects/litecss]). When updating the Snow theme from earlier Tiki versions, for best results remove the older template (.tpl) files and use only the files listed here in the templates/styles/snow directory. Older Snow theme files are incompatible and will cause problems.
