* __Description__: This is an original theme for Tiki CMS (http://tikiwiki.org). It features snowflake-pattern graphic backgrounds for module boxes and other page components, and has an optional Flash title graphic (not tested with the version of the theme). Snow uses the *litecss source-ordered layout method ([http://sourceforge.net/projects/litecss]).
