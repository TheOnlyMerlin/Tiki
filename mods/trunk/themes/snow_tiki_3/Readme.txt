* __Description__: This is an original theme for Tiki CMS (http://tikiwiki.org). It features snowflake-pattern graphic backgrounds for module boxes and other page components, and has an optional Flash title graphic (not tested with the version of the theme). Snow uses the *litecss source-ordered layout method ([http://sourceforge.net/projects/litecss]). When updating the Snow theme from Tiki versions prior to Tiki 3, be sure the older template (.tpl) files are removed and only the files listed here (as part of Snow for Tiki 3) are in the templates/styles/snow directory. Older Snow theme files are incompatible with Tiki 3 and will cause performance problems.
