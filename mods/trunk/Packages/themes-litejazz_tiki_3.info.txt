Description:
Adaptation for TikiWiki CMS/Groupware of the LiteJazz theme for Drupal by Roopletheme (http://roopletheme.com).
Styling is provided for the topbar and bottombar custom code page sections, and there are three color variations, with the red and green versions available as theme options.
