Description:
Adaptation for Tiki Wiki CMS Groupware of the LiteJazz theme for Drupal by Roopletheme (http://roopletheme.com).
Styling is provided for the topbar and bottombar custom code page sections, and there are three color variations, with the red and green versions available as theme options. Recommended module assignments (in this order) -> Top: logo (nobox=y), login_box (nobox=y, mode=popup), menu (nobox=y, type=horiz, id=secondary_site_menu, class=secondary_site_menu); Topbar: search (nobox=y), menu [main "tiki-top" horizontal menu] (nobox=y, id=tiki-top, type=horiz, css=y).
